# Nuxt Monorepo Starter

A minimal Nuxt monorepo template to help you get started fast.

## Use This Template

Scaffold a new project instantly using [giget](https://github.com/unjs/giget):

```bash
# npm
npx giget gh:c5nl3g/nuxt-monorepo-starter my-new-project

# pnpm
pnpm dlx giget gh:c5nl3g/nuxt-monorepo-starter my-new-project

```
